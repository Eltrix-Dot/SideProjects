## 🤖 Dedicated Bots
This folder contains specialized Discord bots, each designed for a specific use case. Unlike all-in-one solutions, these focus on doing one thing exceptionally well:

* **Moderation Bots:** Tools designed to keep servers safe and organized.
* **Utility Bots:** Specialized scripts for specific tasks and workflows.
* **Custom Tools:** One-off bots built for unique community needs.
